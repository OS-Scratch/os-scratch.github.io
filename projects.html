End of support for myOS 1

myOS 1 support ended on November 10, 2024. myOS 1 users should migrate to myOS 2, ScratchOS 1 or myOS 4 Alphas (once available)


Information for myOS 3

Sadly, myOS 3 is not being developed any more.
Why's that? The project was too big and was constantly crashing, disabling me to save my work. myOS 4.0 RC will be released soon (planned on March 2025).

myOS 4.0: The future is here.

myOS 4.0 is coming! The brand-new release with a file system, smart assistant, App Hub, native terminal and more.

Core Functionalities

  • Virtual Machines: VirtualScratch, a built-in software, allows you to effectively browse available OSs and run them.
  • Games: Challenge yourself in a game of "myOS Clicker".
  • FileEdit: view, and edit, files in a number of formats.
  • Aurora Browser: the first browser in myOS that really allows you to take control over your searching experience.
